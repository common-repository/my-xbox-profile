== Description ==

Xbox 360 Gamercard Plugin, that displays your Xbox 360 gamertag details anywhere on your website.

This is the first version of the plugin, its quite simple with a few options but the biggest feature about it is that it is completely customisable using CSS.

Also uses jQuery to display the 5 most recent games played and the amount of achievements. 

== Installation ==

1. Login to your dashboard and under ‘Plugins’ click ‘Add New’.
2. In the serach box type ‘My Xbox Profile’ and then click ‘Search Plugins’.
3. Find the plugin in the list (should be the first one) and click install on the right hand side.
4. When the lightbox window opens click the red ‘Install Now’ button.
5. Once the installation has finished click ‘Activate Plugin’ next to the ‘Actions’ .


== Frequently Asked Questions ==

= How come my recently played games aren't showing? =

Please make sure your Xbox.com Privacy Settings are set to show your recent games. To change the settings please follow the instructions below.

Xbox.com -> My Xbox -> Privacy Settings -> Played Games -> Everyone
